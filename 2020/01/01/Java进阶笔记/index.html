Java进阶笔记

发表于 2020-01-01 分类于 技术笔记

Java进阶

类与对象

  • 对象是实体，需要被创建，可以为我们做事情
  • 类是规范，根据类的定义来创造对象。
  • 封装：把数据和对数据的操作放在一起。
  • 对象变量：对象的管理者。
  • .运算符：操作对象
  • 类定义了对象所具有的变量，这些变量叫做成员变量。
  • 每个对象有自己的变量，和同一个类的其他对象是分开的。
  • 在函数中可以直接写成员变量的名字来访问成员变量
  • 函数是通过对象来调用的。

TIP:this是成员函数的一个特殊的固有的本地变量，它表达了调用这个函数的那个对象。用来区分成员变量和局部变量

  • 定义在函数内部的变量是本地变量，它的生存期和作用域都是函数内部。
  • 成员变量的生存期是对象的生存期，作用域是类内部的成员函数。
  • 成员变量在定义的 地方就可以给出初始值。
  • 没有给出初始值的成员变量会自动获得0值
    • 对象变量的0值表示没有管理任何对象，也可以主动给null值
  • 定义变量初始化可以调用函数，甚至可以使用已经定义的成员变量。
  • 构造函数：如果有一个成员函数的名字和类名完全相同，则在创建这个类的对象的时候会自动调用这个函数。这个函数是不能有返回类型的。
  • 重载：同名但是参数表不同的函数构成重载关系。

对象的交互

  • 封闭的访问属性：
    • private：
      • 只有这个类的内部能够访问。
      • 类内部指定类的成员函数和定义初始化。
      • 这个限制是对类的而不是对对象的。这个私有指的是代码层面的私有，即只要在类的内部进行访问都是被允许的
    • public：
      • 任何人都可以访问
      • 任何人指的是在任何类的函数或定义初始化中可以使用，
      • 使用指的是调用、访问或定义变量。
  • 编译单元：一个Java文件叫做一个编译单元。
  • static变量：类变量，每一个对象都可以拥有这个变量并改变它。（只会在类的装载时进行初始化，与对象的创建无关。）
  • static函数：类函数，不属于对象的函数，而属于类，static之间可以相互调用也可以通过对象调用。

TIP:static字符表示不属于对象而属于类。
